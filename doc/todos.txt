==============================================================================

INSTALLATION                                            *todos-installation*
==============================================================================
Use your favorite plugin manager.

  - {vim-plug}
    1. Add `Plug 'Yahm3/todos.vim'` to your .vimrc
    2. Run `:PlugInstall`


COMMANDS                                                    *todos-commands*
==============================================================================
:Time [args]
    The 'Time' command is used to view the month, day, and time.
    e.g: Jul 21 17:29
    
    Optional arguments:
    d           display day of the month
    t           display current time

:TdGen                                                             *:TdGen*
    Calls `todos#Todo()` to generate or create a new TODO.

:TdOpen                                                           *:TdOpen*
    Calls `todos#Open()` to open the TODOs list.


MAPPINGS                                                    *todos-mappings*
==============================================================================
By default, todos provides the following normal mode mappings:

<leader>td      Generates a new TODO (maps to <Plug>(TodosTodo))
<leader>tdo     Opens the TODO list (maps to <Plug>(TodosOpen))

To disable these default mappings, add the following to your .vimrc: >
    let g:todos_disable_mappings = 1
<
When inside a `todos.txt` buffer, the plugin also automatically maps:

<CR>            Calls `todos#GoTo()` to jump to the selected item.
